# 🧮 Calculator Application

## 📌 Overview

This is a **basic calculator web application** developed using **HTML and JavaScript**.
It allows users to perform simple arithmetic operations in a user-friendly interface.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧾 Two input fields to enter numbers
* ⚡ Instant result display

---

## 🛠️ Technologies Used

* HTML
* JavaScript

---

## 💻 How It Works

1. Enter two numbers in the input fields
2. Choose an operation (Add, Subtract, Multiply, Divide)
3. Click the button
4. View the result instantly
